Snake Game

A game for users to play. Try and get the highest score!

Check out a live demo here!

Tools: Python - pygame

Start

To start the game, press any of the arrow keys and start eating.

Goal

The point of this game is to eat as much as you can without hitting your tail and any walls.

Lost

If you lose, you can choose to try again or quit the application by pressing c and q respectively.

Inspiration

This game was one of my favourites before any other mobile game came out, so I wanted to create a version of my own.
